# Ozon ERP Pro

Ozon SaaS ERP project source archive.

## Contents

- `ozon_saas_erp_source.zip`: complete ERP source code, including backend, frontend, scripts, and CEL shipping-rate workbook.
- Local app URL after running: `http://127.0.0.1:8899`
- Default admin account: ask project owner directly; it is intentionally not shown on the login page.

## Main Features

- Ozon shop binding with Seller API `Client-Id` and `API-Key`
- Manual and scheduled Ozon order sync
- Platform fee / tax rule management
- CEL shipping-rate import from Excel
- Automatic order profit calculation
- Doubao AI assistant configuration

## Run Locally

Extract `ozon_saas_erp_source.zip`, then double-click `start.bat`.
